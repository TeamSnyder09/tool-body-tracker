Documentation Setup

The GitHub Pages site needs to be manually enabled. Follow these steps:

  1. Go to: Repository Settings > Pages
  2. Under "Build and deployment", select Source: Deploy from a branch
  3. Select Branch: main, Folder: / (root)
  4. Click Save
  5. Wait 1-2 minutes for the site to build and deploy

Local Documentation

To view documentation locally:

  • Install: pip install mkdocs mkdocs-material pymdown-extensions
  • Build: mkdocs build
  • Serve: mkdocs serve
  • Visit: http://localhost:8000
